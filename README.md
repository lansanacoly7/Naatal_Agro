# 🌾 Nataal Agro

## 🧠 Intelligent Agricultural Decision Platform

Nataal Agro is an AI-powered agricultural assistant designed to help farmers in Senegal make better decisions using data, AI, weather, and market insights.

---

## 🚀 Vision

Transform agriculture in West Africa by providing farmers with a smart digital co-pilot for decision-making from planting to selling.

---

## 🏗️ Tech Stack

### Mobile
- Flutter

### Web
- React

### Backend
- Django REST Framework

### Database
- PostgreSQL

### AI
- Gemini + Groq

### Notifications
- Firebase

Nataal Agro uses AI to:
